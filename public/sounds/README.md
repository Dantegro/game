# Sounds

Place your background music here.

## Current track
- `lofi-ambient.mp3` — used by the game for menu + world ambience.

## How to add / replace
1. Put a royalty-free lofi (or other vibe) track in this folder.
2. Name it exactly `lofi-ambient.mp3` (or change the default path in `src/audio.ts`).

## Tips
- The game starts the music when you select "Open World" on the menu (this satisfies browser autoplay policies).
- Press **M** to toggle music on/off at any time.
- Volume is set low by default in `src/audio.ts` (you can adjust `DEFAULT_VOLUME`).

## License reminder
Only use tracks you have rights to (Creative Commons, purchased, your own, etc.).
